# 🌠 Live Event Platform: Cinematic Edition

> A premium, personalized entertainment portal designed for the next generation of live digital experiences.

## ✨ The Vision
The Live Event Platform isn't just a streaming site—it's a **Personalized Experience Hub**. We've transitioned from creator-centric metrics to a user-centric activity portal, where your engagement, your visionaries, and your upcoming plans take center stage. 

Designed with a **Glassmorphic Cinematic Aesthetic**, every interaction is crafted to feel premium, alive, and immersive.

---

## 🚀 Key Features

### 👤 My Activity Hub (Personalized Dashboard)
A sophisticated command center for your digital life.
- **Engagement Analytics**: Real-time tracking of events seen, liked, and planned.
- **Interactive Visualization**: Cinematic charts showing your weekly interaction flow.
- **Contribution Tracking**: Centralized view of your discussions and community impact.

### 🌟 Creators Discovery HUB
Find the visionaries behind the stages.
- **Host Index**: Grouped content by organizers, performance artists, and tech leaders.
- **Live Preview Rails**: Horizontal scrolling event previews directly on creator profiles.
- **Verified Status**: Highlighting top-tier contributors in the ecosystem.

### 📼 Interactive Live Streaming
- **4K Glass UI**: A cinematic video player with integrated live chat and dynamic viewer counters.
- **Viewer Retention Tracking**: Passive tracking of "Seen" status after high-quality engagement.
- **Global Reminders**: "Remind Me" functionality with persistent `localStorage` scheduling.

---

## 🎨 Design Philosophy
- **Glassmorphism**: Heavy use of background blurs, multi-layered transparency, and border beams.
- **Cinematic Motion**: Powered by Framer Motion, featuring spring animations, particle bursts, and parallax card effects.
- **Signature Aesthetics**: A HSL-curated color palette (Cyan, Fuchsia, violet) that defines each event category.
- **SSR Optimized**: Built to handle heavy browser-native features (LocalStorage, Viewport APIs) with safe hydration.

---

## 🛠 Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏃‍♂️ Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Pratham-Arun/Live-Event-Platform-VickyBytes.git
   cd Live-Event-Platform-VickyBytes
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Visit [Live Events - VickyBytes](https://lep-vb.vercel.app/) to experience the portal.
